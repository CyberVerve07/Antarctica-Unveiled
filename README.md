# Extreme Explorers - Global Adventure Platform

A comprehensive blogging and exploration platform for the world's most dangerous and beautiful destinations.

## Features
- **Multi-destination coverage**: Amazon Rainforest, Sahara Desert, Mount Everest, Antarctica, Death Valley, and more
- **User blogs & articles**: Write and share your adventure experiences
- **Deep information**: Comprehensive guides, survival tips, and interesting facts
- **Experience sharing**: Connect with fellow explorers and share real-time experiences
- **Rich media**: Image galleries, interactive maps, and stunning visuals
- **Modern UI/UX**: Beautiful design with smooth animations and intuitive navigation

## Tech Stack
- **Frontend**: Next.js 15, React 19, TypeScript
- **Styling**: Tailwind CSS, Framer Motion animations
- **UI Components**: shadcn/ui (Radix UI)
- **Backend**: Firebase (Authentication, Database, Storage)
- **Rich Text**: Tiptap editor for blog writing
- **Icons**: Lucide React

## Destinations Covered
- 🌿 Amazon Rainforest - World's largest tropical rainforest
- 🏜️ Sahara Desert - Largest hot desert on Earth
- 🏔️ Mount Everest - World's highest mountain
- ❄️ Antarctica - Coldest, driest continent
- 🔥 Death Valley - Hottest place in North America
- And many more extreme destinations...

## Getting Started
1. Install dependencies: `npm install`
2. Set up Firebase configuration
3. Run development server: `npm run dev`
4. Open http://localhost:9002

## Project Structure
```
src/
├── app/              # Next.js app directory
├── components/       # Reusable UI components
├── lib/             # Utilities and data

└── types/           # TypeScript type definitions
```
