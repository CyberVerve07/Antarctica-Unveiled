# **App Name**: Antarctica Unveiled

## Core Features:

- Homepage Animation: A 3D animated landscape of Antarctica with snow particle effects on the landing page.
- Informative Content Generation: AI-powered generation of detailed, factual content on various aspects of Antarctica including history, life, research, and climate impact. LLM tool is used to decide what info should go in the content.
- Timeline Creation: Interactive timeline of Antarctic exploration and history.
- Educational Content Structure: Organization of scientific explanations with sub-headings and expandable facts.
- Climate Impact Visualizer: Creation of 3D data representations and diagrams for climate impact sections.
- Navigation and UI: Smooth UI transitions with a clickable, floating navbar, and responsive layout for easy navigation between pages.
- Dynamic Widget Simulation: Simulate a dynamic temperature widget for real-time data.

## Style Guidelines:

- Primary color: Icy Blue (#94BDBD) evokes the cold, pristine environment of Antarctica.
- Background color: Dark navy (#0A0F16), for a stark, winter-themed backdrop.
- Accent color: Soft cyan (#70B8C7), providing a subtle, night-light glow effect to headings and buttons.
- Headline font: 'Space Grotesk' sans-serif with a modern feel, used for headlines and short amounts of body text. Body font: 'Inter', a grotesque-style sans-serif for longer amounts of text. Note: currently only Google Fonts are supported.
- Use minimalist icons related to Antarctic themes (penguins, icebergs, research equipment).
- Dark UI layout with 3D card components, parallax effects, and scrolling animations to engage users.
- Snow particle animations, smooth transitions, hover glow effects, and Lottie animations to enhance user experience.
